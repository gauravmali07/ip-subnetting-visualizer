# IP Subnetting & IPv6 Address Visualizer

A responsive web application for IPv4 subnet calculations and IPv6 address visualization.

## Features
- IPv4 Subnet Calculator
- CIDR & Subnet Mask Conversion
- Network & Broadcast Address Calculation
- Supernetting Support
- IPv6 Compression & Expansion
- IPv6 Validation
- IPv6 Prefix Visualization

## Technologies
- HTML5
- CSS3
- JavaScript
